# Diagnostico GI42001 | Tech Lemonade

App web local-first para aplicar e armazenar diagnosticos da primeira reuniao da ficha Sebraetec GI42001-6.

## Como abrir

Acesse via GitHub Pages ou abra `index.html` no navegador.

## Privacidade

Os dados ficam no navegador do usuario via localStorage. Nenhum dado e enviado para servidor.
